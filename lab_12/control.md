1. Что такое хеширование?

Хеширование - это способ преобразования данных (ключа) в фиксированное число (хеш-код), которое используется для быстрого доступа к данным.

2. Что такое хеш-функция?

Хеш-функция - это функция, которая принимает ключ и возвращает индекс (адрес) в таблице, где будет храниться значение.
Пример:
index = key % size

3. Что называется коллизией?

Коллизия - это ситуация, когда разные ключи дают одинаковый хеш (попадают в одну ячейку таблицы).

4. Какие операции поддерживает хеш-таблица?

Основные операции:
вставка (добавление элемента);
поиск (получение значения по ключу);
удаление элемента;
обновление значения.

5. Почему словарь Python считается примером хеш-таблицы?

Потому что:
данные хранятся в виде ключ–значение;
доступ к элементам происходит через хеширование ключа;
операции выполняются очень быстро (в среднем за O(1)).

6. Что такое ключ и значение?
Ключ - уникальный идентификатор, по которому находится элемент;
Значение - данные, связанные с этим ключом.

Пример:
{"name": "Aliya"}
ключ: "name"
значение: "Aliya"

7. В чем преимущество хеширования перед линейным поиском?
Хеширование: быстрый доступ - O(1) (в среднем)
Линейный поиск: нужно проверять каждый элемент - O(n)

То есть хеширование значительно быстрее при работе с большими данными.

8. Какие методы разрешения коллизий существуют?

Основные методы:
Метод цепочек - в одной ячейке хранится список элементов
Открытая адресация:
линейное пробирование
квадратичное пробирование
двойное хеширование
