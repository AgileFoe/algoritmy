1.Алгоритм - это точное и конечное описание последовательности действий, выполнение которых приводит к решению поставленной задачи.

2.Свойства алгоритма:
Дискретность - алгоритм состоит из отдельных шагов.
Определённость - каждый шаг однозначно определён и не допускает двусмысленности.
Конечность - алгоритм должен завершаться за конечное число шагов.
Результативность - выполнение алгоритма приводит к получению результата.
Массовость - алгоритм применим к целому классу однотипных задач, а не к одному частному случаю.

3.Отличие линейного алгоритма от алгоритма с ветвлением:
Линейный алгоритм - все команды выполняются строго последовательно, без выбора вариантов.
Алгоритм с ветвлением - в зависимости от выполнения условия выбирается один из нескольких возможных путей выполнения алгоритма.

4.Циклы используются для многократного выполнения одного и того же набора команд, когда необходимо повторить действия несколько раз .

5.Типы циклов в программировании:
Цикл с предусловием (while) - проверка условия происходит перед выполнением тела цикла.
Цикл с постусловием (do while) - проверка условия происходит после выполнения тела цикла.
Цикл итератор (for) - используется для перебора значений задонного к итератору.
